unity version 2018.2.14f1
timeline 学习

六种轨道类型:

* Animation Tack:类似于动画编辑器（只能录制和导入动画片段）
* Activation Tack:指定一个游戏物体来控制激活（ugui不行）
* Control Tack:控制多个物体激活
* Autio Tack:声音播放
* Playable
